## Project Structure

```text
src/
├── assets/      # Local static vectors & images
├── components/  # Reusable UI component modules (Hero, Skills, Projects, etc.)
├── context/     # React state providers (ThemeContext)
├── data/        # Static content definitions (portfolioData)
├── pages/       # Custom page view layouts (Home, CaseStudy, Privacy, NotFound)
├── styles/      # Stylesheets and custom tailwind directives (index.css)
└── App.tsx      # Routing and main app structure
```

## Getting Started

Clone the repository

```bash
git clone https://github.com/sanket1035/portfolio.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
